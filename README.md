### 简介

支持展示[知行笔记](https://github.com/xtthaop/zxnote-web)所发布的笔记；  
支持复制链接分享、微信分享；  
支持显示赞赏码；  
接入百度统计；  
……

### 快速开始

**初始化安装依赖**

```
pnpm install
```

**在开发环境运行**

```
pnpm run dev
```

**打包到生产环境**

```
pnpm run build
```
